# Public Variant Overview

The public UMG Envoy Agent package is a compiler-backed OpenClaw plugin with bundled public sample content, public-safe tools, and a controlled runtime-facing bridge into the local UMG validation lane.

`v0.2.2` established the GitHub proof and release-hardening milestone.
`v0.2.3` was the publication-cleanup follow-up.
`v0.3.0-alpha.3` is the minimized public candidate focused on a dedicated public-safe entrypoint, cleaner packaging, and safer shipped bytes before any separate ClawHub publication step.
