# Release Notes - UMG Envoy Agent 0.1.2

UMG Envoy Agent v0.1.2 improves public discovery and listing clarity for the bounded UMG runtime surface. This update adds search-friendly release tags, strengthens benefit-first package language, adds a minimal public workflow example, and clarifies the purpose of matrix-status while preserving the bounded public-safe planner and path-building scope.

## Highlights

- adds explicit discovery-oriented release tags for marketplace search
- strengthens benefit-first package and listing language
- adds a minimal parse -> validate -> render workflow example
- clarifies the role of `matrix-status` in the public lane
- preserves the bounded public-safe planner and path-building scope
